# Review Docs

This folder holds reusable review playbooks and reviewer-facing audit scripts.

Current contents:

- `bitcoin-review-playbook.md` for scoped Bitcoin audits that can be run by
  Codex, Claude, or any other reviewer model
- `bitcoin-review-thread-prompts.md` for ready-to-paste Codex `/review` and
  plain review prompts, one thread per Bitcoin target

Use this folder for reviewer workflows that should stay separate from feature
design docs and implementation-specific audit notes.
