---
title: "Get Profile"
description: "Returns the authenticated user's public profile."
api: "GET http://localhost:8080/api/v1/auth/me"
---

## Behaviour

**`is_locked` vs admin lock** — `is_locked` reflects a self-service OTP brute-force lock that can be cleared via `POST /request-unlock`. A separate admin-initiated lock exists but is intentionally not surfaced here — affected users must contact support.

**`avatar_url` and `last_login_at`** — Both fields are omitted from the response when they have not been set.

## Rate Limiting

**20 requests per 1 minute per IP address**.

Clients that exceed this limit will receive a `429 Too Many Requests` response with a `Retry-After` header indicating when they may retry.

```json 200
{

{
  "code": "rate_limited",

}
```
</ResponseExample>

## Responses

<AccordionGroup>
  <Accordion title="200 — OK">
    The user's public profile. `avatar_url` and `last_login_at` are omitted when not set.

    ```json
    {
      "id": "018f42c3-1a2b-7e4d-9f56-0a1b2c3d4e5f",
      "email": "alice@example.com",
      "display_name": "Alice",
      "avatar_url": "https://cdn.example.com/avatars/alice.png",
      "email_verified": true,
      "is_active": true,
      "is_locked": false,
      "last_login_at": "2026-03-06T10:00:00Z",
      "created_at": "2025-10-01T08:00:00Z"
    }
    ```
  </Accordion>

  <Accordion title="401 — Unauthorized">
    The request is missing a valid JWT or the user ID could not be extracted from the token context.

    ```json
    {
      "code": "unauthorized",
      "message": "missing user id in context"
    }
    ```
  </Accordion>

  <Accordion title="404 — Not Found">
    No user record was found for the ID embedded in the access token. This can occur if the account was deleted after the token was issued.

    ```json
    {
      "code": "not_found",
      "message": "user not found"
    }
    ```
  </Accordion>

  <Accordion title="429 — Rate Limited">
    The IP address has exceeded 20 requests per 1 minute. Check the `Retry-After` response header for how many seconds to wait before retrying.

    ```json
    {
      "code": "rate_limited",
      "message": "too many requests"
    }
    ```
  </Accordion>

  <Accordion title="500 — Internal Server Error">
    An unexpected error occurred while fetching the profile.

    ```json
    {
      "code": "internal_error",
      "message": "internal server error"
    }
    ```
  </Accordion>
</AccordionGroup>
