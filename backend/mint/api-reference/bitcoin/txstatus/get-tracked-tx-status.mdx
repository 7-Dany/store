---
title: "Get Tracked Tx"
description: "Fetch one durable Bitcoin transaction-tracking row by numeric ID."
api: "GET http://localhost:8080/api/v1/bitcoin/tx/{id}"
---

<ParamField path="id" type="integer" required>
  The tracked-transaction ID returned by this API.
</ParamField>

## Behaviour

Returns one durable tx-tracking row owned by the authenticated user. The `id`
path parameter is the tracked-record ID, not a Bitcoin txid.

<ResponseExample>
```json 200
{
  "id": 7,
  "network": "testnet4",
  "tracking_mode": "txid",
  "address": "tb1qfc74wvs6wnz3p2twgza26vukqct4emt2v47xwq",
  "txid": "59a32e91550ce5e1fd33cdd4f2ad8ac33efa3fb9cfac061893d5b73a210a463b",
  "status": "mempool",
  "confirmations": 0,
  "amount_sat": 0,
  "fee_rate_sat_vbyte": 0,
  "first_seen_at": "2026-03-30T16:10:00Z",
  "last_seen_at": "2026-03-30T16:10:00Z",
  "created_at": "2026-03-30T16:10:00Z",
  "updated_at": "2026-03-30T16:10:00Z"
}
```

```json 400
{
  "code": "invalid_id",
  "message": "id must be a positive integer"
}
```

```json 404
{
  "code": "tx_status_not_found",
  "message": "transaction status record was not found"
}
```
</ResponseExample>
