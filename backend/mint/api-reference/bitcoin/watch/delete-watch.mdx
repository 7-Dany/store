---
title: "Delete Watch"
description: "Hard-delete one Bitcoin watch resource."
api: "DELETE http://localhost:8080/api/v1/bitcoin/watch/{id}"
---

<ParamField path="id" type="integer" required>
  The durable watch resource ID.
</ParamField>

## Behaviour

Deletes one watch resource owned by the authenticated user.

This is a hard delete. The deleted watch will no longer appear in future list
results.

## Rate Limiting

**10 requests per 1 minute per IP address**.

<RequestExample>
```json Example
// No request body — the resource ID is passed as a path parameter.
```
</RequestExample>

<ResponseExample>
```json 204
// 204 No Content — empty body
```

```json 404
{
  "code": "watch_not_found",
  "message": "watch resource was not found"
}
```

```json 429
{
  "code": "too_many_requests",
  "message": "too many requests — please slow down"
}
```
</ResponseExample>

## Responses

<AccordionGroup>
  <Accordion title="204 — No Content">
    The watch resource was deleted successfully. No response body is returned.
