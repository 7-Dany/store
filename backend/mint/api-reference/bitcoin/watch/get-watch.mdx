---
title: "Get Watch"
description: "Fetch one Bitcoin watch resource by numeric ID."
api: "GET http://localhost:8080/api/v1/bitcoin/watch/{id}"
---

<ParamField path="id" type="integer" required>
  The durable watch resource ID.
</ParamField>

## Behaviour

Returns one watch resource owned by the authenticated user.

<ResponseExample>
```json 200
{
  "id": 11,
  "network": "testnet4",
  "watch_type": "transaction",
  "txid": "59a32e91550ce5e1fd33cdd4f2ad8ac33efa3fb9cfac061893d5b73a210a463b",
  "status": "active",
  "created_at": "2026-03-31T00:30:00Z",
  "updated_at": "2026-03-31T00:30:00Z"
}
```

```json 404
{
  "code": "watch_not_found",
  "message": "watch resource was not found"
}
```
</ResponseExample>

## Responses

<AccordionGroup>
  <Accordion title="200 — OK">
    The watch resource was found and belongs to the caller.
  </Accordion>

  <Accordion title="400 — Bad Request">
    ```json
    {
      "code": "invalid_id",
      "message": "id must be a positive integer"
    }
    ```
  </Accordion>

  <Accordion title="401 — Unauthorized">
    A valid JWT is required.

    ```json
    {
      "code": "unauthorized",
      "message": "authentication required"
    }
    ```
  </Accordion>

  <Accordion title="404 — Not Found">
    ```json
    {
      "code": "watch_not_found",
      "message": "watch resource was not found"
    }
    ```
  </Accordion>

  <Accordion title="500 — Internal Server Error">
    An unexpected error occurred.

    ```json
    {
      "code": "internal_error",
      "message": "internal server error"
    }
    ```
  </Accordion>
</AccordionGroup>
