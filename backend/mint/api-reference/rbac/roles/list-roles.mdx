---
title: "List Roles"
description: "Returns all active roles in the system."
api: "GET /rbac/roles"
---

<Callout icon="key" color="#FFC107">
  Requires the **`rbac:read`** permission.
</Callout>

## Behaviour

Only active roles are returned. System roles and the owner role are included in the list alongside any custom roles you have created. Roles that have been deleted are not returned.

<RequestExample>
```bash Example
curl -X GET /rbac/roles \
  -H "Authorization: Bearer <token>"
```
</RequestExample>

<ResponseExample>
```json 200
{
  "roles": [
    {
      "id": "018e4c3a-1b2d-7f4e-9c0a-2d3e4f5a6b7c",
      "name": "Store Manager",
      "description": "Can manage products and orders.",
      "is_system_role": false,
      "is_owner_role": false,
      "is_active": true,
      "created_at": "2024-06-01T10:00:00Z"
    }
  ]
}
```

```json 401
{
  "code": "unauthorized",
  "message": "missing or invalid access token"
}
```

```json 403
{
  "code": "forbidden",
  "message": "forbidden"
}
```
</ResponseExample>

## Responses

<AccordionGroup>
  <Accordion title="200 — OK">
    Returns the list of active roles. The array will be empty if no roles exist yet.

    ```json
    {
      "roles": [
        {
          "id": "018e4c3a-1b2d-7f4e-9c0a-2d3e4f5a6b7c",
          "name": "Store Manager",
          "description": "Can manage products and orders.",
          "is_system_role": false,
          "is_owner_role": false,
          "is_active": true,
          "created_at": "2024-06-01T10:00:00Z"
        }
      ]
    }
    ```
  </Accordion>

  <Accordion title="401 — Unauthorized">
    The request is missing a valid access token.

    ```json
    {
      "code": "unauthorized",
      "message": "missing or invalid access token"
    }
    ```
  </Accordion>

  <Accordion title="403 — Forbidden">
    The authenticated user does not hold the `rbac:read` permission.

    ```json
    {
      "code": "forbidden",
      "message": "forbidden"
    }
    ```
  </Accordion>

  <Accordion title="500 — Internal Server Error">
    An unexpected error occurred on the server.

    ```json
    {
      "code": "internal_error",
      "message": "internal server error"
    }
    ```
  </Accordion>
</AccordionGroup>
