---
title: "Remove User Role"
description: "Removes the active role assignment for a given user."
api: "DELETE http://localhost:8080/admin/rbac/users/{user_id}/role"
---

<ParamField path="user_id" type="string" required>
  The UUID of the user whose role assignment should be removed.
</ParamField>

## Behaviour

Removes the target user's current role assignment. After removal the user has no role and will be denied access to any permission-guarded endpoints.

**Self-removal is not allowed.** You cannot remove your own role assignment.

**Owner role assignments cannot be removed through this endpoint.** If the target user holds the owner role, the request is rejected regardless of who is making it.

**The last active owner cannot be removed.** At least one active owner must exist at all times. Attempting to remove the sole remaining owner will be rejected to prevent the system from becoming unmanageable.

```json Example
{}
```
</RequestExample>

## Responses

<AccordionGroup>
  <Accordion title="204 — No Content">
    The role assignment was successfully removed. The user now has no active role.
  </Accordion>

  <Accordion title="401 — Unauthorized">
    The request is missing a valid access token.

    ```json
    {
      "code": "unauthorized",
      "message": "missing or invalid access token"
    }
    ```
  </Accordion>

  <Accordion title="403 — Forbidden">
    The authenticated user does not hold the `rbac:manage` permission.

    ```json
    {
      "code": "forbidden",
      "message": "forbidden"
    }
    ```
  </Accordion>

  <Accordion title="404 — Not Found">
    The user has no active role assignment to remove.

    ```json
    {
      "code": "user_role_not_found",
      "message": "user has no active role assignment"
    }
    ```
  </Accordion>

  <Accordion title="409 — Conflict">
    Returned in three cases: attempting to remove your own role, attempting to remove an owner role via this endpoint, or attempting to remove the last active owner in the system.

    ```json
    {
      "code": "cannot_modify_own_role",
      "message": "you cannot modify your own role assignment"
    }
    ```

    ```json
    {
      "code": "cannot_reassign_owner",
      "message": "owner role cannot be reassigned via this route"
    }
    ```

    ```json
    {
      "code": "last_owner_removal",
      "message": "cannot remove the last active owner"
    }
    ```
  </Accordion>

  <Accordion title="500 — Internal Server Error">
    An unexpected error occurred on the server.

    ```json
    {
      "code": "internal_error",
      "message": "internal server error"
    }
    ```
  </Accordion>
</AccordionGroup>
